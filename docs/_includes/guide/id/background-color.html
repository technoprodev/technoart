# Background Color Warna background pada Technoart dibagi menjadi 2 bagian, yaitu grayscale background dan full color background.
## Grayscale Background Grayscale background merupakan background dengan warna hitam putih. Grayscale background cocok dipakai untuk kalimat agar nyaman dibaca oleh user. Grayscale background memiliki 9 pilihan warna, yaitu: {% capture option_code %}
.bg-lightest
.bg-lighter
.bg-light
---
.bg-grayest
.bg-grayer
.bg-gray
---
.bg-darkest
.bg-darker
.bg-dark
{% endcapture %} {% include option-code.html content=option_code %}
## Hover Grayscale Background Dengan hover, style grayscale background hanya akan terlihat jika kita menggeser pointer ke komponen. Sama seperti grayscale background, hover grayscale background memiliki 9 pilihan warna, yaitu: {% capture option_code %}
.hover-bg-lightest
.hover-bg-lighter
.hover-bg-light
---
.hover-bg-grayest
.hover-bg-grayer
.hover-bg-gray
---
.hover-bg-darkest
.hover-bg-darker
.hover-bg-dark
{% endcapture %} {% include option-code.html content=option_code %}
## Full Color Background Full color background merupakan background yang bewarna warni. Full color background cocok dipakai untuk kalimat yang menarik perhaian user (branding).
