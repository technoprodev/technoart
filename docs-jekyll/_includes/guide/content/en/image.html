# Image Secara default, komponen img ditampilkan sesuai dengan ukuran aslinya. Misalnya, image.jpg dengan ukuran asli 300x300 akan ditampilkan pada halaman web dengan ukuran yang sama yaitu 300px x 300px. Masalah akan muncul ketika img ditampilkan pada layar mobile, yaitu img terlihat terlalu besar melebihi ukuran parent containernya. Solusi untuk masalah tersebut adalah, atur lebar img menjadi 100%. Lebar img akan menjadi sama dengan parent containernya, sehingga ketika img ditampilkan pada layar mobile, ukurannya akan menyesuaikan lebar parentnya.
