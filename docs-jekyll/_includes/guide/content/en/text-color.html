# Text Color Warna text pada Technoart dibagi menjadi 2 bagian, yaitu grayscale text dan full color text.
## Grayscale Text Grayscale text merupakan text dengan warna hitam putih. Grayscale text cocok dipakai untuk kalimat agar nyaman dibaca oleh user. Grayscale text memiliki 9 pilihan warna, yaitu: {% capture option_code %}
.text-lightest
- bg-dark
.text-lighter
- bg-dark
.text-light
- bg-dark ---
.text-grayest
.text-grayer
.text-gray
---
.text-darkest
.text-darker
.text-dark
{% endcapture %} {% include option-code.html content=option_code %} Warna text default yang digunakan Technoart adalah .text-grayest.
## Hover Grayscale Text Dengan hover, style grayscale text hanya akan terlihat jika kita menggeser pointer ke komponen. Sama seperti grayscale text, hover grayscale text memiliki 9 pilihan warna, yaitu: {% capture option_code %}
.hover-text-lightest
.hover-text-lighter
.hover-text-light
---
.hover-text-grayest
.hover-text-grayer
.hover-text-gray
---
.hover-text-darkest
.hover-text-darker
.hover-text-dark
{% endcapture %} {% include option-code.html content=option_code %}
